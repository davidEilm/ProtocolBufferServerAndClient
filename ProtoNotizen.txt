Installation
    C# -> NuGet Package Manager Google.Protobuf installieren
    
    GO -> go get -u github.com/golang/protobuf/protoc-gen-go

Wie wird ein .proto file "konvertiert/kompiliert(?)"?
    protoc --go_out=. Person.proto
